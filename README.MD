#Тестовое задание

## Installation
Для сборки необходимо выполнить команду:
```
mvn clean install
```

Перед запуском необходимо в файле application.properties указать параметры для подключения к БД

После сборкт проекта необходиом запустить файл run

По дефолту будет обрабатываться 1000000 записей, чтобы обработать кастомное количество записей 
необходимо открыть файл run любым текстовым редактором и в конце строки main-jar-with-dependencies.jar 
через пробел добавить число.
